github使用

功能强大的$github$,可托管项目代码等(好像码云也可以
基于$github$的博客在一位朋友的帮助在$linux$系统上搭好了

属实弟弟

$git$语句
$touch$ 文件名 创建文件
$git$ $init$ 初始化仓库
$git$ $status$ 查看状态
$git$ $add$ 文件名 (提交到暂存区)
$git$ $commit$ $-m$ “描述” (提交到仓库)
$vim$ 文件名 修改文件
$rm$ 文件名 删除文件

$1$.初始化用户名和邮箱
$git$ $config$ –$global$ $user.name$ $’ZJA0811’$
$git$ $config$ –$global$ $user.email$ '1246785974@qq.com‘

上传步骤
1.在$github$上新建一个仓库,创建时有一个$https$地址,记录此地址,后面用
2.打开$git$ $bash$,转到你所要上传的文件夹目录下,并输入$git$ $init$
3.将项目添加到仓库中去:$git$ $add$,如果添加某个文件,可以使用$git$ $add$ $xxx$
4.将添加的文件提交到仓库:$git$ $commit$ $-m$ “–注释–”
5.将仓库关联到$github$:$git$ $remote$ $add$ $origin$ https://xxxx,$https$为刚才$github$上创建仓库的地址
6.把文件推送到$github$仓库:$git$ $push$ $-u$ $origin$ $master$,下次推送时,可以把$-u$去掉

如何在$README.md$里展示图片呢？
1.首先把图片提交到$github$,记录该图片所在的地址:$https://xxx$
2.在$README.md$中添加图片:！

如$git$ $push$出现错误,则设置权限等方式解决
