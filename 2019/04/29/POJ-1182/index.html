POJ-1182

POJ 1182

食物链

$Time Limit: 1000MS \qquad \qquad \qquad \qquad Memory Limit: 10000K$

Description

动物王国中有三类动物$A$，$B$，$C$，这三类动物的食物链构成了有趣的环形。$A$吃$B$，$B$吃$C$，$C$吃$A$。

现有$N$个动物，以$1-N$编号。每个动物都是$A$，$B$，$C$中的一种，但是我们并不知道它到底是哪一种。

有人用两种说法对这$N$个动物所构成的食物链关系进行描述：

第一种说法是1 X Y，表示$X$和$Y$是同类。

第二种说法是2 X Y，表示$X$吃$Y$。

此人对$N$个动物，用上述两种说法，一句接一句地说出$K$句话，这$K$句话有的是真的，有的是假的。当一句话满足下列三条之一时，这句话就是假话，否则就是真话。

  1. 当前的话与前面的某些真的话冲突，就是假话；

  2. 当前的话中$X$或$Y$比$N$大，就是假话；

  3. 当前的话表示$X$吃$X$，就是假话。

你的任务是根据给定的$N(1 \leq N \leq 50,000)$和$K(0 \leq K \leq 100,000)$句话，输出假话的总数。

Input

第一行是两个整数$N$和$K$，以一个空格分隔。

以下$K$行每行是三个正整数$D$，$X$，$Y$，两数之间用一个空格隔开，其中$D$表示说法的种类。

若$D=1$，则表示$X$和$Y$是同类。

若$D=2$，则表示$X$吃$Y$。

Output

只有一个整数，表示假话的数目。

Sample Input

100 7
1 101 1
2 1 2
2 2 3
2 3 3
1 1 3
2 3 1
1 5 5

Sample Output

3

Source

NOI 2001

题解

裸的并查集题目，其实感觉应该不太会出，并查集主要用来优化时间复杂度之类的吧

解法0

种类并查集，开一个$rank$数组记录动物之间的关系，$pre$数组记录祖先，通过向量推导动物之间的关系，(这里就不赘述了，请移步这里)（hhhhh感觉自己说话好直男啊

解法1

图论拆点思想，其实给我的感觉就是把每种情况都列出来，$X$和$Y$都只有$A$,$B$,$C$三种可能。

$D=1$表示同类，当$D=1$时是假话的可能只有两种情况：

  1. $X$属于$A$而$Y$属于$B$（这种情况和$X$属于$B$而$Y$属于$C$是一样的）

  2. $X$属于$A$而$Y$属于$C$

其他情况就把$X$和$Y$合并起来。

$D=2$表示捕食，当$D=2$时是假话的可能也只有两种情况：

  1. $X$属于$A$而$Y$属于$A$（这种情况和$X$属于$B$而$Y$属于$B$相同）

  2. $X$属于$A$而$Y$属于$C$（即与题目给出的$C$吃$A$不同）

否则合并其他情况。

（这种方法很好但是有可能会被卡空间，因为要开一个三倍大于解法0的数组

这个细节很重要 当时WA了一次 三倍的数组 不过好像样例不开三倍数组过不了QAQ

详情请见《挑战程序设计竞赛》84页
