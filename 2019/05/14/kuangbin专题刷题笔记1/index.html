把题目嘴巴一下

$BFS$：主函数输入输出+一些初始化
$BFS$函数：两个点 $current$ $next$ 标记 入队 非空时 取出点 搜索操作 剪枝 标记 继续入队

$problem A$.棋盘问题
走过一列就把它标记下来下次的时候就不可以再摆放在这一列
（因为题目要求不可以将棋子摆放在同一行和同一列）
然后就从下一行开始寻找可行的地方，直到我们摆放的棋子数与我们被要求摆放的棋子数相同时，我们就将方案数进行一次++，然后在进行递归下去

$problem B$. Dungeon Master
$BFS$从二维转到三维 往六个方向进行搜索
$queue$用法 （$FIFO$）
$push()$ //即插入元素
$pop（)$ //删除元素
$front()$ //读取队首元素
$back()$ //读取队尾元素
$empty（）$ //判断队列是否为空
$size（）$ //返回队列大小
注意全局变量和$memset$使用 队列使用要熟练
（代码虽然跑得慢 但毕竟是亲生的

$problem C$. Catch the cow
一个点进入队列不同于一个节点（其实也差不多 爆了$1$次$RE$…

$problem E$. Find The Multiple
给定一个数$n$，求用$0,1$组成的十进制数$m$，$m$将$n$整除
双口$BFS$+同余模定理

$problem F$.Prime Path
数位$BFS$,四位数转化为在四维空间上的搜索
只有素数进行搜索 分离数位对每一位进行搜索

$problem G.$Shuffle’m Up
模拟水题 用$map…$

$problem I.Fire Game$
两点$BFS$，求双搜最优解

$problem K$. 迷宫问题
$BFS$和$DFS$好像都可以（好像是数据不强没有卡掉$DFS$ 加个标志记录路径

$problem L$.oil deposits
用$DFS$判断连通块 往八个方向进行搜索

$problem M$.非常可乐
解法$0$.数论知识 （其实我现在也还不懂
先咕在这 参考一篇blog
解法$1$.$BFS$
有明确的初始状态和结束状态，每次操作有$6$种情况…慢慢写

$problem N.$ Find a way
从两个点开始$BFS$,记录遇到每个$KFC$的步数，暴力枚举出最短时间
